# 🌊 IoT-Based Smart Drain Monitoring & Emergency Response System

## 📌 Project Overview
The IoT-Based Smart Drain Monitoring & Emergency Response System is a smart city solution designed to monitor drainage conditions in real time, detect blockages, harmful gases, abnormal water flow, and potential flood situations. This system helps improve public safety, supports early disaster prevention, and enables efficient emergency coordination.

---

## 🚀 Features
- 🌐 Real-time drainage monitoring using IoT sensors  
- 📏 Water level detection using Ultrasonic Sensor  
- ☣️ Harmful gas detection using MQ-135 Gas Sensor  
- 💧 Flow monitoring for blockage identification  
- 📡 ESP32-based data transmission  
- 🖥️ Full-stack web dashboard for complaint and task management  
- 👥 Multi-role access for Citizens, Corporation, and Emergency Teams  
- 🚨 Emergency alerts for flood risk and hazardous conditions

## 📊 Workflow
1. Sensors collect real-time drainage data  
2. ESP32 processes and transmits data  
3. Backend stores data in MongoDB  
4. Web dashboard displays analytics  
5. Authorities receive alerts and take action
